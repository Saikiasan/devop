Program VS Process

Programs Processes
1 Programs contain a set of instructions designed to compute a specific task. It is an instance of an executing program.
2 It is a passive entity as it resides in the secondary memory. It is an active entity as it is created during execution and loaded into the main memory.
3 It exits at a single phase and continues to exist until it is deleted. It exists for a limited span of time as it gets terminated after the completion of the task.
4 A program is a static entity A process is a dynamic entity
5 It dosen't have any control block. Has its own control block called Process Control Block (PCB)
6 Has two logical components: code and data. In addition to program data, process also requires additional information required for the management and execution.
